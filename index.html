Tentang Saya

Saya merupakan mahasiswa Teknik Informatika yang aktif mengerjakan berbagai proyek berbasis web sebagai bentuk pengembangan kemampuan dan penerapan ilmu yang telah dipelajari. Beberapa proyek yang pernah saya kerjakan antara lain website kasir untuk UMKM, website rumah sakit sebagai media informasi dan layanan digital, serta sistem pengajuan fasilitas kampus berbasis web.

Selain pengembangan sistem, saya juga memiliki ketertarikan pada desain antarmuka (UI/UX). Saya menyukai desain yang bersih, modern, dan mudah digunakan karena dapat meningkatkan kenyamanan pengguna dalam berinteraksi dengan sebuah aplikasi atau website. Melalui berbagai proyek tersebut, saya terus berusaha meningkatkan kemampuan teknis dan kreativitas dalam bidang pengembangan web.

Project Saya

Website Kasir

Website Kasir merupakan aplikasi berbasis web yang digunakan untuk membantu proses transaksi penjualan pada toko dan UMKM. Sistem ini menyediakan fitur pencatatan barang, pengelolaan stok, serta perhitungan total belanja secara otomatis. Dengan adanya website kasir ini, proses transaksi menjadi lebih cepat, rapi, dan mengurangi kesalahan pencatatan manual.

Website Rumah Sakit

Website Rumah Sakit dibuat sebagai media informasi dan layanan digital bagi pasien. Website ini menampilkan profil rumah sakit, informasi layanan, jadwal dokter, serta fitur pendaftaran pasien secara online. Tujuan dari sistem ini adalah untuk memudahkan pasien dalam mendapatkan informasi dan mengurangi antrean pendaftaran di rumah sakit.

Pengajuan Fasilitas Kampus

Sistem Pengajuan Fasilitas Kampus merupakan aplikasi berbasis web yang digunakan oleh mahasiswa dan dosen untuk mengajukan peminjaman atau permohonan fasilitas kampus secara online. Sistem ini membantu pihak kampus dalam mengelola data pengajuan, memantau status permohonan, serta meningkatkan efisiensi dan transparansi dalam proses administrasi fasilitas.
